# Installation

## Requirements

- Python 3.13 or higher

## Install from PyPI

```bash
pip install bundlebase
```

## Install with Poetry

```bash
poetry add bundlebase
```

## Install for Jupyter Notebooks

For Jupyter notebooks, use the sync API with the jupyter extra:

```bash
pip install "bundlebase[jupyter]"
```
